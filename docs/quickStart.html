Quick Start: 5 steps to run PURE on an HTML page

Here are the very, very... easy steps to start using PURE, and discover a totally new way of building web interfaces.
First have a look at the examples.
Nothing flashy in those examples yet. Their purpose is mainly didactic.

1. Download PURE

Go to download page and get the last version available.
Copy the /pure directory somewhere on your web server or computer.

2. Add the JS files

Place a script tag in the <head> of your document with the src pointing to the location of the js/pure.js file 
<script type="text/javascript" src="../js/pure.js"></script>
Place a script tag in the <head> of your document with the src pointing to a JS library of your choice (only jQuery is supported at this time and is provided within the /pure/js directory) 
<script type="text/javascript" src="../js/jqueryMin.js"></script>

3. Build your HTML

Put the HTML below in the <body> section of your document 
<div id="hello">
  Hello <span class="who">World</span>
</div>

<a href="#" onclick="render();return false;">Render</a>

<div id="target"></div>

4. Add the Javascript

Place the function below in the <head> section of your page 
function render(){
  var html = document.getElementById('hello');
  var context = { "name": "Mary" };
  var directive = {"span.who":"name"};
  html = $p.map( directive, html);
  var target = document.getElementById('target');
  $p.render(html, context, target);
}

5. Render it

Click the link Render in your page. You should see Hello Mary appearing below the link you just cliked.

Optional

If you plan to use the examples as tutorials, you may find the css useful, so add them in the head section of your page like below:
Either: 
<link href="white.css" rel="stylesheet" type="text/css" title="white"/>
Or: 
<link href="black.css" rel="stylesheet" type="text/css" title="black"/>

if you plan to use PURE on Internet Explorer

You need to add the namespace pure to the HTML tag of your page, otherwise you may encounter some strange behaviours. 
<html xmlns="http://www.w3.org/1999/xhtml" xmlns:pure="http://beebole.com/pure">
