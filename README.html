HIGHLIGHTS

*  Build and render your templates with HTML, CSS and JavaScript, nothing else.
* Unobtrusive, the HTML remains absolutely untouched.
* As a result, any part of your HTML document can become a template.
* Fast, templates are compiled from HTML to Javascript for blazing rendering speed.
* Runs on the browser side.
* JSON data support.
* Can be used as a jQuery plugin.
* Open Source, and released under the MIT License. It means you can use it and
   modify it as much as you want and where you want.
